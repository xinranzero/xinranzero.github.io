【Java基础】面向对象进阶：static、继承、单例

静态（static）

修饰成员变量或成员方法。

static修饰成员变量

按照有无static修饰，成员变量分为：

  1. 有–>类变量（静态变量）。属于类，在计算机中只有一份，被类的全部对象共享。通过类名调用：类名.静态变量。
  2. 无–>实例变量。属于每个对象。
